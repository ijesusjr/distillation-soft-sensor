# Distillation Column Soft-Sensor

Machine learning-based soft-sensor for predicting ethanol concentration (purity) in distillation columns.

## Features

- Real-time purity prediction using XGBoost
- Feature importance visualization
- Model performance metrics
- User-friendly web interface

## Model Performance

- R² Score: 0.9998
- RMSE: 0.0010
- MAE: 0.0008

## Usage

1. Set process variables in the sidebar
2. Click "Make Prediction"
3. View predicted purity and status
4. Check feature importance for explanations

## Installation (Local)
```bash
pip install -r requirements.txt
streamlit run app.py
